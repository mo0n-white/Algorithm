### 문제 설명

<p>N개의 수 A_1, A_2, ..., A_N과 L이 주어진다.</p>

$A_{i-L+1}$ ~ $A_i$ 중 최솟값을 $D_i$라고 할 때 D에 저장된 수를 출력하는 프로그램을 작성하시오.

<p>이때 i≤0인 A_i는 무시하고 D를 구해야 한다.</p>

### 입력 

 <p>첫째 줄에 N과 L이 주어진다. (1 ≤ L ≤ N ≤ 5,000,000)</p>

 <p>둘째 줄에 N개의 수 A_i가 주어진다. (-10<sup>9</sup> ≤ A_i ≤ 10<sup>9</sup>)</p>

### 출력 

 <p>첫째 줄에 D_i를 공백으로 구분해 순서대로 출력한다.</p>
